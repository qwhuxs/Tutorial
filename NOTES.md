\## 🛠 Розділ 1: Створення Next.js проєкту та запуск сервера розробки



\### 🔹 Встановлення pnpm та створення проєкту



\*\*Опис:\*\* Для оптимізації роботи було обрано менеджер пакетів `pnpm`, який працює швидше та ефективніше за стандартні npm чи yarn.



1\.  \*\*Встановлення pnpm:\*\*

&nbsp;   ```bash

&nbsp;   npm install -g pnpm

&nbsp;   ```

2\.  \*\*Створення нового Next.js проєкту:\*\* Проєкт ініціалізовано на основі офіційного навчального шаблону:

&nbsp;   ```bash

&nbsp;   npx create-next-app@latest nextjs-dashboard --example "\[https://github.com/vercel/next-learn/tree/main/dashboard/starter-example](https://github.com/vercel/next-learn/tree/main/dashboard/starter-example)" --use-pnpm

&nbsp;   ```

3\.  \*\*Підготовка середовища:\*\* Проєкт відкрито у \*\*Visual Studio Code\*\* для подальшої роботи та перевірки структури файлів.

\### 🔹 Встановлення залежностей та запуск сервера



\*\*Опис:\*\* Після створення структури необхідно завантажити модулі та перевірити працездатність локального сервера.



\* \*\*Встановлення пакетів:\*\*

&nbsp;   ```bash

&nbsp;   pnpm i

&nbsp;   ```

\* \*\*Запуск сервера розробки:\*\*

&nbsp;   ```bash

&nbsp;   pnpm dev

&nbsp;   ```



\*\*Результат:\*\* Сервер успішно запущено на порту \*\*3000\*\*. При переході за адресою \[http://localhost:3000](http://localhost:3000) відображається стартова сторінка (на даному етапі без фінальних стилів).
